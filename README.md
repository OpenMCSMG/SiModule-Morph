# Blockbuster 变身模块指令
变身模块是一个指令，用于智能修复普通变身插件带来的崩溃问题。它基于 `BlockBuster` 短篇长剧拍摄工具模组实现。

## 指令格式
/playmorph [玩家] [模型] [nbt]

- **[玩家]**: 指定变身的玩家用户名。
- **[模型]**: 指定变身使用的模型。
- **[nbt]**: (可选) 指定额外的 NBT 数据。
